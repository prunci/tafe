HTML STRUCTURES

The structure of a basic html pageis as follows


  1. DOCTYPE!
  2. head
  3. title
  4. title
  5. body
  6. body

Opening tags < > almost always require a closing tag </>

  • <h1> to <h6> can be used to change test size h1 is the largest
  • br inserts white space under text - no closing tag required
  • p creates a paragraph
  • div creates a larger container for related groups of paragraphs, allows styling to be added, can also contain images
  • strong makes text bold
  • em make text italic
  • span creates a space in a line of text- doesnt seem to work
  • ol ordered list ordered list create a list of bullets item using li
  • ul unordered list creates a list of number item using li
  • li list item used inside ol and ul
  • img to insert an image. It can use a / to close the tag -no closing tag required
  • video to insert a video. It requires a closing tag
  • href to insert a hyperlink to another webpage or ??????

Attribute

Attribute can be added to an element. An attribute has a Name and a Value

  • id in the form id="value"
  • scr= used with img and video to provide location of the media
  • alt= used with img to provide a description of the image incase the image is not there or cannot be seen
  • width= used with video to size the window
  • height= used with video to size the window
  • controls adds play and volume controls
  • a an anchor or pointer used with href
